# 🌍 Travio – Travel Booking Interface

Travio is a responsive travel booking interface built with **HTML, CSS, and JavaScript**.  
It provides a modern front‑end design where users can explore destinations, check availability, and simulate booking journeys.  
This project focuses purely on the client side, showcasing interactive UI elements, smooth transitions, and accessibility‑friendly layouts.

---

## ✨ Features
- Responsive layout for desktop and mobile
- Interactive booking form with dynamic elements
- Destination highlights and clean card‑style design
- Smooth transitions and modern UI styling
- Built with pure front‑end technologies (HTML, CSS, JS)
